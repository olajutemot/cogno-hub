what will you learn?

Week 1

Introduction to front-end web development
  • understading front-end web development
  • overview of front-end technologies HTML, CSS, javaScript
  • setting up a development environment
  • Basic Git command and version control

Week 2

HTML and CSS
  • HTML document structure and syntax
  • Semantic HTML and accessibility
  • Basic css syntax and selectors
  • CSS Box model and layout
  • Responsive design with media queries

Week 3

javaScript Fundamentals
  • Introduction to javaScript
  • Data types and variables
  • Conditional statement and loops
  • Functions and scope
  • DOM manipulation and events

Week 4

Advanced javaScript
  • Object-oriented programming in javaScript
  • Array methods and higher order function
  • Asynchronous programming and callbacks
  • Promises and async/await
  • Handling errors and debugging

Week 5

Frameworks and librariest
  • Introduction to front-end frameworks(React, vue, Angular)
  • Creating components and routing in a framework
  • Introduction to CSS frameworks (Bootstrap, Foundation)
  • Working with APIs and consuming data

Week 6

Advanced Front-End Techniques
  • Animations and transitions with CSS and javaScript
  • Accessibility considerations for front-end development
  • Web performance optimization
  • Cross-border compatibility and troubleshooting
  • Best practice for front-end development
